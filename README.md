# ExamBrowser

## Cara mengecek apakah menggunakan Exam Browser menggunakan Javascript
```javascript
if (typeof(CefSharp) !== "object") {
	alert("Menggunakan Exam Browser");
}
else {
	alert("Tidak Menggunakan Exam Browser");
}
```

## Cara mengubah URL
1. Buka file settings.xml
2. Ubah isi dari Url
3. Selesai
